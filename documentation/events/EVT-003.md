Emitted when copy-from-default-branch semantics trigger on first checkout of a new branch.
The KB store from the resolved default branch is copied to `.kb/branches/<new-branch>/`.

The default branch is determined in this order:
1. `.kb/config.json` `defaultBranch` (if set)
2. `origin/HEAD` (if available)
3. `main` (fallback only if neither of the above is set)
