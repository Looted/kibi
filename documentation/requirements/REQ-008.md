`kibi init --hooks` installs `post-checkout` and `post-merge` git hooks that
keep the KB in sync with the working tree. On branch checkout, if the branch KB does not exist, it is created by copying from the resolved default branch (see brief.md for precedence: config `defaultBranch`, then `origin/HEAD`, then `main`).
